# Manual Actions

- Заповнити УДК на титульній сторінці.
- Заповнити ПІБ студента і номер групи.
- Заповнити ПІБ, науковий ступінь і посаду керівника.
- Проставити службові підписи, дати й реквізити.
- Якщо до здачі буде реально проведено педагогічний експеримент, внести тільки фактичні результати без вигаданих чисел.
