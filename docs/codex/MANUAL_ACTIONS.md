# Manual Actions

- Підтвердити службовий блок на титулі:
  - ПІБ студента;
  - групу;
  - ПІБ/посаду/ступінь керівника;
  - ПІБ консультанта з нормоконтролю;
  - дату й живі підписи, якщо цього вимагає кафедра.
- За потреби додати офіційний УДК, якщо він має бути у фінальному поданні.
- Якщо фінальний експорт або друк робитиметься саме в Microsoft Word, виконати одну контрольну візуальну звірку сторінок змісту після локальної репагінації в цільовому середовищі.
- Якщо кафедра вимагає іншу фінальну PDF-машину, повторно звірити лише титул, сторінки 7-8, сторінки з рисунками 3.1/3.2/3.3/4.1 і додаток А.
