# Manual Actions

- Заповнити УДК на титульній сторінці.
- Заповнити ПІБ студента і номер групи.
- Заповнити ПІБ, науковий ступінь і посаду керівника.
- Внести кафедральні реквізити, підписи, дати та службові поля, які не можна визначити автоматично.
- Якщо кафедра вимагає окрему службову сторінку за внутрішнім шаблоном, перенести туди лише офіційні реквізити без зміни змісту основного тексту.
- Не додавати емпіричні результати, якщо педагогічний експеримент реально не проводився.
