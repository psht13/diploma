# Ручне втручання

Нижче лишено тільки ті пункти, які не можна достовірно завершити автоматично без даних автора роботи або кафедри. Після автоматичних виправлень цей список буде ще раз звужено.

## Титульні та службові реквізити

- [ПОТРІБНО РУЧНЕ ВТРУЧАННЯ: уточнити УДК, якщо його не буде визначено достовірно].
- [ПОТРІБНО РУЧНЕ ВТРУЧАННЯ: уточнити точну офіційну назву типу документа на титулі].
- [ПОТРІБНО РУЧНЕ ВТРУЧАННЯ: вказати ПІБ студента].
- [ПОТРІБНО РУЧНЕ ВТРУЧАННЯ: вказати номер групи].
- [ПОТРІБНО РУЧНЕ ВТРУЧАННЯ: вказати ПІБ наукового керівника, посаду, науковий ступінь].
- [ПОТРІБНО РУЧНЕ ВТРУЧАННЯ: проставити підписи, службові реквізити та дати].
- [ПОТРІБНО РУЧНЕ ВТРУЧАННЯ: підтвердити, чи потрібні офіційні сторінки «Завдання» / «Календарний план» саме для магістерської курсової роботи].

## Експериментальні дані

- [ПОТРІБНО РУЧНЕ ВТРУЧАННЯ: додати реальні результати педагогічного експерименту].
- [ПОТРІБНО РУЧНЕ ВТРУЧАННЯ: додати статистичний аналіз тільки після отримання реальних даних].
- [ПОТРІБНО РУЧНЕ ВТРУЧАННЯ: за потреби виконати ручний browser-based E2E-прогін з установленим `ollama` і локальною моделлю `llama3.1:8b`].
