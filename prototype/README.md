# JS Tutor Prototype

Мінімальний vertical slice для курсової роботи:

- генерація вправи з базового JavaScript;
- написання або редагування розв'язку;
- запуск тестів у `Web Worker`;
- адаптивний фідбек у стилі репетитора;
- інтеграція з локальним `Ollama`, якщо він доступний;
- детермінований резервний режим, якщо Ollama недоступна або повертає невалідний JSON;
- мінімальний JSON-експорт сесії для майбутніх A/B або crossover досліджень.

## Запуск

1. За потреби підготуйте локальну модель:

```bash
ollama pull llama3.1:8b
ollama serve
```

2. Запустіть локальний статичний сервер із кореня репозиторію:

```bash
npm run serve
```

3. Відкрийте:

```text
http://localhost:4173/prototype/
```

## Зауваження

- Весь користувацький JavaScript виконується лише у браузері, у `Web Worker`, із таймаутом.
- Поточний worker є лише посиленим sandbox для MVP, але не повноцінною безпековою пісочницею. Він обмежує частину глобалів, перевіряє `functionName` і має timeout/error handling, проте не повинен розглядатися як строгий security boundary.
- Серверне виконання або `eval` на сервері відсутні.
- Якщо `Ollama` недоступна, застосунок переходить у резервний режим і використовує вбудовані вправи та шаблонний фідбек.
- Кнопка `Пояснити тему` вмикає явний `interactionMode: "explain"`, тому explanation-flow більше не залежить лише від regex у тексті запиту.
- Кнопка `Експортувати сесію JSON` зберігає тему, складність, `attemptsCount`, першу помилку, останню дію, transcript, фінальний статус запуску та часову позначку.
